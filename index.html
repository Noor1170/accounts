Configuration

Need help? See the Setup Guide below. Your credentials are saved only in your browser.

📋 Setup Guide (one-time)

1. Go to console.cloud.google.com → New Project.
2. Enable Google Sheets API.
3. Create OAuth 2.0 credentials → Web app → add http://localhost and the URL you use to open this file as authorized JS origins. Paste the Client ID above.
4. Create a new Google Sheet. Copy its ID from the URL (the long string between /d/ and /edit). Paste the Sheet ID above.
5. Click Sign in with Google — the app will create the required columns automatically.
